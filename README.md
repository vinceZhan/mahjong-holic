# Mahjong Holic 🀄

A structured collection of Mahjong rules, variations, and learning resources — with a focus on both **classic Mahjong** and **Sichuan Mahjong (四川麻将)**.

---

## 📌 About

This repository is designed to:

* Provide **clear and beginner-friendly explanations** of Mahjong
* Separate **general mechanics** from **regional rule variations**
* Serve as a **reference hub** for learning and comparing Mahjong styles
* Potentially support future **code implementations or tools**

---

## 📚 Documentation

### 🧠 Basics (Start Here)

* 📄 `docs/mahjong-basics.md`
  Covers the **fundamentals of Mahjong**, including:

  * Tiles and suits
  * Game setup
  * Turn flow
  * Basic winning structure

---

### 🀄 Rule Variations

#### Classic Mahjong

* 📄 `docs/classic-mahjong.md` 
  A general “standard” version of Mahjong, including:

  * Common rules and gameplay
  * Melds (Chow / Pong / Kong)
  * Winning conditions
  * Basic scoring concepts

#### Sichuan Mahjong (四川麻将)

* 📄 `docs/sichuan-mahjong.md`
  A popular regional variant known for fast-paced gameplay:

  * Ding Que (定缺)
  * Blood Flow rules (血流成河)
  * Continuous wins
  * Simplified tile set (variation-dependent)
